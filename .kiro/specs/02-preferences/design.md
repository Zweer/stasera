# Preferences Design

## Onboarding Quiz

### Layout (Mobile-first)
- Full-screen flow, no distractions (no navbar)
- Two cards stacked vertically (mobile) or side by side (desktop)
- Each card: event illustration/icon + title + short description + vibe tags
- Tap to select → card animates (scale up, glow) → reason options appear

### Reason Selection
- After picking a card, 3-4 pill buttons slide in from bottom
- Each pill is a reason: "Perché è all'aperto", "Perché mi piace il jazz"
- Tap a reason → next round auto-advances with smooth transition

### Progress
- Minimal dot indicators at the top (filled = completed)
- "Round 3 of 8" subtitle text, muted

### Completion
- Celebration micro-animation (confetti or sparkle)
- "Your profile is ready!" → Show summary of top 3 preferences
- CTA: "See what's on this weekend" → navigate to recommendations

## Profile View Page

### Layout
- Tag cloud or horizontal bar chart showing preference strengths
- Tags color-coded by category (music=purple, setting=green, vibe=amber...)
- Each tag tappable for explanation: "Jazz: +0.8 — you chose it 4 times"
- "Refine" button → triggers 3 more comparison rounds

### Visual Style
- Warm, personal feel — this is "your taste"
- Use gradients matching the evening/night theme of the app
- Dark card on dark background with subtle glow borders
